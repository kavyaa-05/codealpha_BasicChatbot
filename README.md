## 📌 Project Title
**Basic Chatbot using Python (CodeAlpha Task 4)**


## 📖 Description
This project is a simple rule-based chatbot developed using Python as part of the CodeAlpha internship. The chatbot interacts with users by responding to predefined inputs such as greetings, basic questions, and exit commands.
It demonstrates how fundamental programming concepts can be used to simulate a basic conversation system. The chatbot continuously runs in a loop, allowing real-time interaction until the user chooses to exit.


## ✨ Features
* 🤖 Interactive chatbot system
* 💬 Responds to common user inputs (hello, how are you, bye, etc.)
* 🔁 Continuous conversation using loops
* 🧠 Uses if-elif conditions for decision making
* ⚡ Beginner-friendly and easy to understand
* 🎯 Handles unknown inputs with default responses


## ▶️ How to Run the Project
1. Make sure Python is installed on your system
2. Clone this repository or download the files
3. Open the project folder in VS Code or any IDE
